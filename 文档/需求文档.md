# 2026 FIFA World Cup Global Info Hub
### 项目基础信息
- 项目名称：2026 FIFA World Cup Global Info Hub
- 产品类型：海外世界杯资讯数据站
- 面向用户：全球英文互联网用户
- 合规约束：纯资讯/工具属性，无博彩、无现金竞猜、无付费充值、无赛事直播、无版权侵权素材、无用户登录/注册体系
- 核心目的：Google SEO流量获取 + Google AdSense广告变现 + 亚马逊联盟分销带货
- 部署环境：Vercel 一键部署，无备案、无国内服务器、无后端依赖

### 指定技术栈
Next.js + Tailwind CSS + Axios
纯前端渲染，无数据库、无后台管理、无用户体系
全站响应式适配PC/移动端，支持暗黑/亮色主题切换

### 全站页面清单 + 功能明细
#### 1. 首页 `/`
- 顶部固定导航：Home, Schedule, Scores, Standings, Teams, News, Polls, Gear
- 首页大屏世界杯开赛倒计时组件
- 今日焦点赛事置顶展示模块
- 近3日赛事列表预览
- 最新完赛赛果滚动通告
- 底部全局页脚：Privacy Policy, Disclaimer, Contact, Subscribe
- 预埋广告位：顶部通栏、中部信息流、底部固定栏

#### 2. 完整赛程 `/schedule`
- 按日期筛选/切换赛事列表
- 赛事展示：主队+客队+队徽、本地时区自动换算时间、比赛阶段、比赛状态
- 状态标识：Upcoming / Live (Delayed) / Full Time
- 列表间隙预留广告位

#### 3. 实时比分 & 赛果 `/scores`
- 今日赛事延迟比分展示，3分钟自动前端刷新
- 按时间倒序排列，完赛场次优先展示
- 极简赛果数据：比分、关键事件极简记录
- 禁止展示：赔率、盘口、投注、让球相关内容

#### 4. 小组积分榜 `/standings`
- 按小组拆分展示，自动按规则排序
- 展示字段：Rank, Team, Pld, W, D, L, GF, GA, GD, Pts
- 出线资格高亮标记
- 移动端横向滚动适配

#### 5. 球队资料 `/teams` + `/teams/[teamId]`
- 全部参赛球队卡片列表
- 球队详情：队徽、所属小组、AI英文简介、关联全部赛程
- 通用无版权视觉素材，禁止使用FIFA官方版权资源

#### 6. 资讯文章 `/news` + `/news/[articleSlug]`
- 文章列表：标题、发布时间、摘要、分类标签
- 文章详情：正文、相关文章推荐
- SEO伪静态链接，正文内嵌+底部广告位
- 支持分类：Previews, Reviews, Team Facts, Player Focus

#### 7. 强制合规页面
- 隐私政策 `/privacy`
- 免责声明 `/disclaimer`
- 404错误页面 `/404`

#### 8. 预留迭代模块（基础版完整包含）
- 球迷投票 `/polls`：单题投票、实时结果展示、防重复提交
- 球迷周边推荐 `/gear`：联盟商品卡片、跳转外链
- 邮件订阅 `/subscribe`：极简订阅入口

### SEO & 性能要求
- 每页独立配置 Title、Meta Description、规范H1-H6层级
- 图片懒加载、ALT英文描述、代码压缩优化
- 自动生成 sitemap.xml，URL语义化小写横杠分隔
- 面包屑导航、移动端友好适配
- 页面极速加载，降低跳出率

### UI/交互规范
- 设计风格：极简现代体育风，清爽无冗余动效
- 主色配置：深蓝 `#0F172A`、强调绿 `#10B981`
- 全局支持暗黑/亮色模式一键切换
- 广告不遮挡核心内容，无诱导点击、无恶意弹窗

### 数据接口规范
- 预留 TheSportsDB / Football-Data.org API 对接代码
- 封装统一请求方法，支持定时刷新、失败兜底
- 仅使用公开结构化赛事数据，不抓取版权内容
- 无用户数据存储、无敏感权限获取

### 交付要求
- 完整可直接部署的整站源码
- 目录结构规范，代码简洁可读
- 所有页面、路由、广告位、API预留全部实现
- 无需额外二次修改，下载即可编译部署
