# 2026 FIFA World Cup Global Info Hub
### 项目基础信息
- 项目名称：2026 FIFA World Cup Global Info Hub
- 产品类型：海外世界杯资讯数据官网
- 展示语言：**整站所有页面、导航、按钮、文案、提示文字、资讯内容全部为英文展示，前端无任何中文**
- 面向用户：全球英文互联网足球球迷
- 合规约束：纯资讯工具属性，无博彩、无现金竞猜、无付费充值、无赛事盗播、不使用FIFA官方版权素材、无用户登录注册体系
- 核心目的：获取Google SEO自然流量、Google AdSense广告变现、亚马逊联盟周边带货
- 部署方式：Vercel免费一键部署，无需备案、无需国内服务器、无后端无数据库

### 指定技术栈
Next.js + Tailwind CSS + Axios
纯前端静态渲染，不做后台管理系统、不做会员体系
全站自适应PC/平板/移动端，支持亮色/暗黑模式一键切换

### 全站页面清单 & 功能明细
#### 1. 首页 `/`
- 顶部固定导航栏：Home、Schedule、Scores、Standings、Teams、News、Polls、Gear
- 世界杯主题开赛倒计时大屏组件
- 今日焦点赛事置顶模块
- 近3日赛事列表预览
- 最新完赛赛果滚动展示
- 底部页脚固定链接：Privacy Policy、Disclaimer、Contact、Subscribe
- 预埋广告位：顶部通栏、中部信息流、底部固定广告栏

#### 2. 完整赛程页面 `/schedule`
- 支持按日期筛选、今日/明日/全部赛事切换
- 展示内容：对阵两队+队徽、自动适配本地时区比赛时间、赛事阶段、赛事状态
- 状态标注：Upcoming / Live (Delayed) / Full Time
- 赛事列表间隙预留广告位

#### 3. 比分赛果页面 `/scores`
- 延迟比分展示，前端每3分钟自动刷新
- 按时间倒序排列，已完赛赛事优先
- 只展示极简赛果、比分、关键赛事记录
- 严禁出现赔率、盘口、投注、竞猜相关内容

#### 4. 小组积分榜页面 `/standings`
- 按小组分栏展示，自动按世界杯规则排序
- 展示字段：Rank、Team、Pld、W、D、L、GF、GA、GD、Pts
- 小组出线队伍高亮标记
- 移动端支持表格横向滚动适配

#### 5. 球队列表+详情页 `/teams`、`/teams/[teamId]`
- 全部参赛球队卡片列表
- 球队详情：队徽、所属小组、英文简介、关联该球队全部赛程
- 使用无版权通用队徽素材，规避官方版权风险

#### 6. 资讯文章页面 `/news`、`/news/[articleSlug]`
- 文章列表：标题、发布时间、摘要、分类标签
- 文章详情页：英文正文、相关文章推荐
- SEO伪静态链接，正文内嵌广告+底部广告位
- 资讯分类：Previews、Reviews、Team Facts、Player Focus

#### 7. 合规必备页面
- 隐私政策页 `/privacy`（全英文）
- 免责声明页 `/disclaimer`（全英文）
- 404错误页面 `/404`

#### 8. 预留迭代模块（一期直接建好框架）
- 球迷投票 `/polls`：单题投票、实时结果图表、设备防重复投票
- 周边推荐 `/gear`：亚马逊联盟商品卡片，点击外链跳转
- 邮件订阅 `/subscribe`：极简邮箱订阅入口

### SEO & 性能要求
- 每个页面独立配置Title、Meta Description，H1-H6标题层级规范
- 图片懒加载、全部图片配置英文Alt描述、代码压缩
- 自动生成sitemap.xml，URL语义化、小写横杠分隔
- 全站配置面包屑导航，适配Google移动端收录标准
- 页面加载速度优化，降低用户跳出率

### UI 设计规范
- 设计风格：极简现代体育风，无多余花哨动效
- 主色调：深蓝 #0F172A、强调色荧光绿 #10B981
- 全局支持亮色/暗黑模式切换
- 广告位布局合理，不遮挡核心内容，无恶意弹窗、无诱导点击

### 数据接口规范
- 预留 TheSportsDB / Football-Data.org 免费API对接代码
- 统一封装请求方法，支持定时刷新、请求失败兜底占位
- 只调用公开结构化赛事数据，不爬取版权资讯内容
- 不存储用户隐私数据，无敏感权限获取

### 开发交付要求
- 输出完整可直接部署的整站源码
- 项目目录结构规范，代码简洁带基础注释
- 所有页面、路由、广告位、API对接预留全部完成
- 下载后可直接编译、部署到Vercel，无需二次大改
